# ScanFactory filter domains

Тестовое задание от компании scan-factory 

#### Задача
Написать скрипт на Python, который для каждого проекта (project_id), на основе существующих данных, создает регулярное
выражение, которое будет отсеивать "мусорные" домены. И записывает его в таблицу rules.

#### Запуск
Для запуска необходимо отредактировать путь до базы данных (sqlite3) в скрипте.
Который находится в переменной ```DATABASE_PATH = './domains.db'```

После этого запустить скрипт командой ```python3 main.py```
